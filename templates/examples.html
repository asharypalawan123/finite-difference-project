Step-by-Step Solutions

Three fully worked examples — one for each method — showing exactly how the approximation is computed from inputs to derivative.

At a Glance
Method Formula Uses Points Accuracy Best For
Forward [f(x+h) − f(x)] / h x, x+h O(h) Fast / simple cases
Backward [f(x) − f(x−h)] / h x−h, x O(h) Real-time / sequential data

Example 01 Forward Difference

f(x) = x²  at  x = 2, h = 0.1

Forward

Given

f(x)x²
x2
h0.1
Formula[f(x+h) − f(x)] / h
\[ f'(x) \approx \frac{f(x+h) - f(x)}{h} \]

Step-by-Step

  • 1 f(x+h) = f(2.1) = (2.1)² = 4.41
  • 2 f(x) = f(2) = (2)² = 4.00
  • 3 Numerator = 4.41 − 4.00 = 0.41
  • 4 Derivative = 0.41 / 0.1 = 4.1
True derivative: f'(x) = 2x → f'(2) = 4.0  |  Error ≈ 0.1

True value: 4.0
Absolute error: 0.1
Example 02 Backward Difference

f(x) = x³  at  x = 3, h = 0.1

Backward

Given

f(x)x³
x3
h0.1
Formula[f(x) − f(x−h)] / h
\[ f'(x) \approx \frac{f(x) - f(x-h)}{h} \]

Step-by-Step

  • 1 f(x) = f(3) = 3³ = 27.000
  • 2 f(x−h) = f(2.9) = (2.9)³ = 24.389
  • 3 Numerator = 27.000 − 24.389 = 2.611
  • 4 Derivative = 2.611 / 0.1 = 26.11
True derivative: f'(x) = 3x² → f'(3) = 27.0  |  Error ≈ 0.89
Approximate f'(3)
26.110000
True value: 27.0
Absolute error: 0.89
Example 03 Central Difference

f(x) = sin(x)  at  x = 1, h = 0.01

Central

Given

f(x)sin(x)
x1
h0.01
Formula[f(x+h) − f(x−h)] / 2h
\[ f'(x) \approx \frac{f(x+h) - f(x-h)}{2h} \]

Step-by-Step

  • 1 f(x+h) = sin(1.01) ≈ 0.846832
  • 2 f(x−h) = sin(0.99) ≈ 0.836026
  • 3 Numerator = 0.846832 − 0.836026 = 0.010806
  • 4 Derivative = 0.010806 / (2 × 0.01) = 0.540302
True derivative: f'(x) = cos(x) → f'(1) = cos(1) ≈ 0.540302
Approximate f'(1)
0.540302
True value: 0.540302
Error ≈ 0.000000 (excellent!)
