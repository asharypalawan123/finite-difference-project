Finite Difference Methods

Numerical techniques that approximate derivatives using discrete function values — fundamental tools in scientific computing, engineering, and applied mathematics.

Background

Finite Difference Methods replace the limiting process of calculus with a finite step size \(h\), yielding computable approximations that converge to the true derivative as \(h \to 0\). They are widely used when the exact derivative is difficult, costly, or impossible to obtain analytically.

All three methods stem from the Taylor series expansion of \(f(x)\). By choosing which neighboring points to include, we trade simplicity for accuracy.

Methods
f '₊

Forward Difference

Estimates the derivative using the current point and the next nearby point. Derived directly from the limit definition of the derivative.

\[ f'(x) \approx \frac{f(x+h) - f(x)}{h} \]

Simple and efficient. Ideal for fast approximations where high accuracy is not critical. Only looks forward in the domain.

O(h) — first order
f '₋

Backward Difference

Uses the current point and the previous nearby point. Useful in real-time or sequential systems where future data is unavailable.

\[ f'(x) \approx \frac{f(x) - f(x-h)}{h} \]

Same accuracy order as forward difference. Sensitivity to step size applies equally to both methods.

O(h) — first order
f '₀

Central Difference

Uses points on both sides of the target. The symmetric structure cancels out odd-order error terms, yielding smaller truncation error.

\[ f'(x) \approx \frac{f(x+h) - f(x-h)}{2h} \]

Preferred for numerical computations requiring precision. Halving \(h\) reduces error by a factor of 4.

O(h²) — second order
Step Size & Error

h Choosing Step Size

The step size \(h\) controls the trade-off between truncation error and floating-point rounding error. Neither extreme is ideal.

  • LARGE h Poor approximation — truncation error dominates.
  • TINY h Floating-point rounding errors accumulate dangerously.
  • MODERATE h Best numerical stability — balances both error sources.

ε Truncation Error Orders

The error order describes how rapidly the approximation improves as \(h\) shrinks. Higher order = faster convergence.

Forward Difference \(O(h)\)
Backward Difference \(O(h)\)
Central Difference \(O(h^2)\)

Central difference converges quadratically.

Advantages & Limitations

Advantages

  • Simple and easy to implement
  • Useful when analytical derivatives are unavailable
  • Applicable to discrete experimental data
  • Computationally very efficient
  • Widely used in engineering and simulations

Limitations

  • Produces approximation (truncation) errors
  • Highly sensitive to choice of step size
  • Very small \(h\) introduces rounding errors
  • Less accurate than symbolic differentiation
