01 Overview

Finite Difference Methods are numerical techniques used to approximate derivatives of functions using nearby discrete function values — instead of analytical differentiation. They are fundamental in scientific computing when the exact derivative is difficult, costly, or impossible to obtain.

These methods work by replacing the limiting process of calculus with a finite step size \(h\), yielding computable approximations that converge to the true derivative as \(h \to 0\).

02 Methods
f '₊

Forward Difference

Estimates the derivative using the current point and the next nearby point. Derived directly from the limit definition of the derivative.

\[ f'(x) \approx \frac{f(x+h) - f(x)}{h} \]

Simple and computationally efficient. Ideal for fast approximations where high accuracy is not critical.

O(h) — first order
f '₋

Backward Difference

Uses the current point and the previous nearby point. Especially useful in real-time systems where future data is unavailable.

\[ f'(x) \approx \frac{f(x) - f(x-h)}{h} \]

Shares the same accuracy order as forward difference. Sensitivity to step size applies equally.

O(h) — first order
f '₀

Central Difference

Uses points on both sides of the target value. The truncation error is smaller, making it the most accurate of the three methods.

\[ f'(x) \approx \frac{f(x+h) - f(x-h)}{2h} \]

Preferred choice for numerical computations where precision matters.

O(h²) — second order
03 Step Size & Error Analysis

h Importance of Step Size

The step size \(h\) critically controls the trade-off between truncation error and floating-point rounding error.

  • LARGE h Poor approximation — truncation error dominates.
  • TINY h Floating-point rounding errors accumulate dangerously.
  • MODERATE h Best numerical stability — balances both error sources.

ε Truncation Error Orders

Each method has a characteristic truncation error order that determines how rapidly the approximation improves as \(h\) decreases.

Forward Difference \(O(h)\)
Backward Difference \(O(h)\)
Central Difference \(O(h^2)\)

Central difference converges quadratically — halving \(h\) reduces its error by a factor of 4.

04 Advantages & Limitations

Advantages

  • Simple and easy to implement
  • Useful when analytical derivatives are unavailable
  • Applicable to discrete experimental data
  • Computationally very efficient
  • Widely used in engineering and simulations

Limitations

  • Produces approximation (truncation) errors
  • Highly sensitive to choice of step size
  • Very small \(h\) introduces rounding errors
  • Less accurate than symbolic differentiation
